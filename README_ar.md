## أفضل 20 Best Grok Prompts

هذه هي أكثر grok prompts فعالية وتنوعاً من مجموعتنا. كل منها يعمل عبر مستويات متعددة من Grok ويقدم نتائج عالية الجودة باستمرار.

### 1. تحليل الويب في الوقت الفعلي
```
حلل المواضيع الرائجة الحالية على [المنصة/الموضوع] في الوقت الفعلي. قدّم:
- أهم 5 عناصر رائجة مع السياق
- لماذا كل منها رائج (مدعوم بالبيانات)
- تحليل المشاعر
- تنبؤ بـ 24 ساعة القادمة
التنسيق كإحاطة إخبارية.
```
**لماذا يعمل:** يستفيد من وصول Grok للبيانات الفورية - غير متوفر في ChatGPT
**يعمل مع:** Grok Premium، Spicy Mode

### 2. خبير تصحيح الأكواد
```
أنت مساعد تصحيح أكواد من النخبة. سأقدم كوداً به خطأ. ردك:
1. حدد الخطأ بالضبط (رقم السطر)
2. اشرح لماذا يفشل
3. قدم الكود المصحح
4. اقترح طريقتين لمنع أخطاء مماثلة

[الصق كودك هنا]
```
**لماذا يعمل:** المخرجات المنظمة تضمن تصحيحاً شاملاً
**يعمل مع:** جميع مستويات Grok

### 3. مولد سلاسل X/Twitter
```
أنشئ سلسلة X فيروسية عن [الموضوع] مع:
- تغريدة جذابة (بحد أقصى 280 حرفاً، توقف التمرير)
- 8-12 تغريدة مليئة بالقيمة
- كل تغريدة: فكرة رئيسية واحدة
- تغريدة نهائية مع دعوة لاتخاذ إجراء
- تضمين هاشتاغات ذات صلة
التنسيق: تغريدة 1/12، تغريدة 2/12، إلخ.
```
**لماذا يعمل:** محسّن لتكامل Grok مع X والميكانيكا الفيروسية
**يعمل مع:** جميع مستويات Grok

- الإضاءة: [الساعة الذهبية/استوديو/درامية/إلخ]
- زاوية الكاميرا: [لقطة واسعة/لقطة قريبة/منظر جوي/إلخ]

   - المفاهيم الأساسية للتعلم

### 1. تستفيد من نقاط قوة Grok الفريدة
على عكس ChatGPT، يتفوق Grok في:
- **الوصول للبيانات الفورية**: الأحداث الجارية، المواضيع الرائجة، بيانات السوق المباشرة
- **التكامل مع X/Twitter**: فهم عميق لديناميكيات وسائل التواصل الاجتماعي
- **توليد الصور**: نموذج FLUX المدمج لصور عالية الجودة
- **توليد الفيديو**: قدرات إنشاء فيديو أصلية
- **الوضع بدون رقابة**: Spicy Mode للتحليل الجدلي/غير المفلتر

**مثال:** بدلاً من عام "حلل هذا الموضوع"، استخدم "حلل بيانات الاتجاهات الحالية على X لـ [الموضوع] من آخر 6 ساعات."

### 2. توفر بنية واضحة
يؤدي Grok أفضل أداء مع prompts منظمة:

**سيء:**
```
اكتب شيئاً عن التسويق
```

**جيد:**
```
أنشئ استراتيجية تسويق مع:
1. تحليل الجمهور المستهدف
2. 3 توصيات للقنوات
3. تخصيص الميزانية
4. مؤشرات الأداء الرئيسية للتتبع
5. جدول زمني 90 يوماً
```

### 3. تحدد تنسيق المخرجات
أخبر Grok بالضبط كيف تريد النتائج:
- "التنسيق كجدول"
- "اكتب في نقاط"
- "أنشئ دليل خطوة بخطوة"
- "ولّد مخرجات JSON"
- "استخدم عناوين markdown"

### 4. تتضمن السياق والقيود
- **الدور**: "تصرف كـ [نوع الخبير]"
- **الجمهور**: "اشرح لـ [مبتدئ/خبير]"
- **الطول**: "احتفظ به تحت [X] كلمة"
- **النبرة**: "استخدم نبرة [احترافية/عادية/فكاهية]"
- **القيود**: "تجنب [أشياء محددة]"

### 5. تستخدم أمثلة عند الإمكان
أظهر لـ Grok ما تريد:
```
ولّد 3 عناوين رئيسية مثل هذا المثال:
مثال: "كيف [X] زاد الإيرادات 300% في 90 يوماً بدون إعلانات مدفوعة"

للموضوع: [موضوعك]
```

### 6. تكرر وتربط الـ Prompts
للمهام المعقدة، قسّم إلى خطوات:
```
الخطوة 1: "اسرد 10 أفكار محتوى لـ [الموضوع]"
الخطوة 2: "وسّع الفكرة #3 إلى مخطط كامل"
الخطوة 3: "اكتب قسم المقدمة"
```

**تعلم المزيد:** تحقق من دليل [هندسة الـ Prompts لـ Grok](guides/prompt-engineering-for-grok.md).

## Grok vs ChatGPT Prompts

ليست كل الـ prompts تعمل بالتساوي عبر نماذج AI. إليك كيفية التحسين لـ Grok:

| الميزة | ChatGPT Prompts | Grok Prompts | الفائز |
|---------|-----------------|--------------|--------|
| **البيانات الفورية** | محدودة (قطع التدريب) | وصول كامل للويب الحالي | Grok |
| **وسائل التواصل الاجتماعي** | فهم عام | تكامل عميق مع X | Grok |
| **توليد الصور** | يتطلب API DALL-E | نموذج FLUX مدمج | Grok |
| **إنشاء الفيديو** | غير متوفر | توليد فيديو أصلي | Grok |
| **وضع بدون رقابة** | مفلتر بشدة | Spicy Mode متوفر | Grok |
| **عمق التفكير** | ممتاز | جيد جداً | ChatGPT |
| **توليد الكود** | ممتاز | جيد جداً | ChatGPT |
| **محتوى طويل** | ممتاز | جيد جداً | ChatGPT |

### أمثلة تكييف الـ Prompts

**Prompt ChatGPT:**
```
حلل تأثير العمل عن بعد على الإنتاجية
```

**Prompt Grok المحسّن:**
```
باستخدام بيانات 2024 الحالية، حلل اتجاهات إنتاجية العمل عن بعد:
- أحدث الدراسات من هذا العام
- المشاعر الفورية من مناقشات X
- سياسات الشركات الحالية (شركات التكنولوجيا الكبرى)
- معدلات اعتماد أدوات الإنتاجية
- تنبؤات للسنتين القادمتين
استشهد بالمصادر مع التواريخ.
```

**لماذا نسخة Grok أفضل:** تستفيد من البيانات الفورية، تكامل X والحداثة.

**اقرأ المقارنة الكاملة:** [دليل Grok vs ChatGPT Prompts](guides/grok-vs-chatgpt-prompts.md)

## الأسئلة الشائعة

### ما هو Grok AI؟
Grok هو مساعد AI أنشأته xAI (شركة AI الخاصة بإيلون ماسك). لديه وصول ويب فوري، تكامل X/Twitter وقدرات توليد صور/فيديو مدمجة.

### ما هو Spicy Mode؟
Spicy Mode هو وضع Grok بدون رقابة الذي يوفر استجابات مباشرة وغير مفلترة بدون حواجز الشركات. متوفر في Grok Premium.

### هل هذه الـ prompts مجانية للاستخدام؟
نعم، جميع الـ prompts في هذا المستودع مرخصة بـ MIT. استخدمها كما تريد - شخصياً أو تجارياً.

### هل تعمل مع Grok Free؟
معظم الـ prompts تعمل مع Grok Free. الـ prompts التي تتطلب بيانات فورية، توليد صور أو Spicy Mode تحتاج Grok Premium.

### كم مرة يتم تحديث الـ prompts؟
نضيف grok prompts جديدة أسبوعياً. ضع نجمة على المستودع لتلقي إشعارات التحديثات.

### هل يمكنني المساهمة بـ prompts خاصة بي؟
بالتأكيد! قدّم عبر [GitHub Issues](https://github.com/grok-spicy-mode/best-grok-prompts/issues). نراجع جميع التقديمات.

### أين يمكنني العثور على المزيد من الـ prompts؟
زر **[GrokPrompts.app](https://grokprompts.app)** لقاعدة بيانات قابلة للبحث تحتوي على أكثر من 1000 grok prompts عبر جميع الفئات.

### كيف تقارن مع prompts ChatGPT؟
الـ grok prompts محسّنة لنقاط قوة Grok (البيانات الفورية، تكامل X، توليد الصور/الفيديو). راجع [دليل المقارنة](guides/grok-vs-chatgpt-prompts.md).
