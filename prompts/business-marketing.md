## 2. Competitor SWOT Deep Dive

**Category:** Competitive Intelligence
**Difficulty:** ⭐⭐

> Analyze [competitor name] using SWOT framework. Strengths: 1) Core competencies, 2) Competitive advantages, 3) Resources and assets, 4) Brand positioning. **Why it works:** SWOT structure is familiar and comprehensive. Competitive action plan transforms analysis into strategy. Weakness exploitation and strength defense create tactical clarity.

## Bonus: Business & Marketing Prompt Best Practices

**Maximize Business Value from Grok:**
- **Be specific with context**: Industry, company size, target market, competitive landscape
- **Request frameworks**: Marketing models (4Ps, AIDA, Jobs-to-be-Done) provide structure
- **Ask for alternatives**: Multiple options reveal trade-offs and enable better decisions
- **Demand metrics**: KPIs, benchmarks, and measurement frameworks make strategies accountable

**Common Business Prompt Patterns:**
- "Analyze [situation] using [framework]. Provide [specific outputs] with [success criteria]"
- "Create [deliverable] for [audience] achieving [goal] within [constraints]"
- "Compare [option A] vs [option B] across [dimensions]. Recommend with rationale"
- "Develop [strategy type] including [required elements] measured by [KPIs]"

**Iteration Strategies:**
- Start broad: "Market overview for X" → Then narrow: "Deep dive on segment Y"
- Request refinement: "Make this more [specific/aggressive/conservative/creative]"
- Test alternatives: "Generate 3 different approaches to solving [problem]"
- Validate assumptions: "What could go wrong with this strategy? Stress test it"

**Integration with Business Tools:**
- Export to spreadsheets: Request table formats for data analysis
- Create presentations: Ask for slide-by-slide content outlines
- Build documents: Generate report structures with sections and subsections
- Email/communication: Request copy at different formality levels
